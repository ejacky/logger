Pagon Logger
============

[![Build Status](https://travis-ci.org/pagon/logger.png)](https://travis-ci.org/pagon/logger)

非常简单的日志库，支持debug, info, warn, error, critical等日志等级，也支持以stream的方式添加handler

依赖
============

- PHP 5.3.9+
- Composer

库依赖：

`pagon/fiber`
